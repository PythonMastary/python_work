## 環境

## 開発手順
1. 問題に取り掛かるとき，developブランチを切る(*1)  
2. 各問題に対してコミットする  
3. 問題を解いたら，プルリクを出す  
4. 大きな区切りができたら，masterにプルリクを出してマージする

例)  
'''
(master)$ git checkout -b develop
'''
